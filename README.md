# tech-blog

## Purpose
A WordPress style blog site where users can add posts, as well as comment and vote on other's posts.

## Built With
* JavaScript
* HandleBars
* Express
* MySQL

## Usage
Install the necessary NPM packages in package.json, run node server from your terminal.
